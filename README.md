# SaveMate - Shopping & Social Hub

A comprehensive shopping and social platform for South African consumers.

## Features

- **Deal Discovery**: Browse products from major South African retailers
- **Social Feed**: Share deals and shopping experiences
- **Shopping Lists**: Create and manage your shopping lists
- **Black Market**: Job listings and service offerings
- **User Profiles**: Personal profiles with followers/following

## Tech Stack

- **Frontend**: Vanilla JavaScript with ES6 modules
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Deployment**: Vercel
- **Styling**: CSS3 with CSS Variables for theming

## Setup

1. Clone the repository
2. Run `npm install`
3. Create `.env` file with your Supabase credentials
4. Run `npm run dev` for development
5. Run `npm run build` for production

## Environment Variables

Create a `.env` file:
\`\`\`
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
\`\`\`
